# Git Commands

Initialize the git repository
```
git init
```

Add a remote to Github
```
git remote add origin https://xxxxxxx
```

Show Git Status
```
git status
``` 

Add files to staging area
```
git add -A
```

Commit Staged files
```
git commit -m "Initial Commit"
```

Push to Github

```
git push origin master
```
